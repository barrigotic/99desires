99 DESIRES

99 Desires is bucket list software written in PHP, CSS and Jquery.
To use it you need to have configure you PC as a Php server for example with xampp or MAMP.

How to use it.
1) Unzip the main package and move the folder in a directory in your php server.
2) Open you favorite web browser, go to the project page on your php server and open index.php.
3) Enjoy
