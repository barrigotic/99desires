99 DESIDERI

99 Desideri è un programma scritto in Php, CSS e Jquery per gestire una lista di desideri.
Per utilizzarlo è necessario aver configurato il proprio computer come server PHP ad esempio con Xampp o MAMP.

Come usarlo.
1) Una volta scompattato lo zip sposta tutta la cartella in una cartella a tua scelta nel server Php.
2) Apri il tuo browser preferito e vai all'indirizzo dove hai caricato il progetto e apri index.php.
3) Divertiti!
